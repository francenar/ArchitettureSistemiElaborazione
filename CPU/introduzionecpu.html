Introduzione alla CPU

La CPU (nota come unità di elaborazione centrale), è colloquialmente conosciuta come processore. È il sottosistema che implementa la maggioranza delle operazioni fondamentali di un computer e coordina l'esecuzione delle operazioni tra i sottosistemi periferici. La maggior parte delle CPU moderne è realizzata sotto forma di microprocessore digitale, che utilizza un'architettura a registri generali.

Una CPU ha il compito di eseguire le istruzioni di un programma presente nella ROM in fase di boot, per poi passare alla memoria centrale (come la RAM) per l'esecuzione. Le architetture di CPU più comuni sono quelle basate su:

  • Von Neumann: dove i dati e le istruzioni risiedono nella stessa memoria.
  • Harvard: in cui i dati e le istruzioni sono separati in due memorie distinte, migliorando le prestazioni.

La CPU esegue le istruzioni in diverse fasi:

  • FETCH: preleva l'istruzione dall'indirizzo nel registro PC.
  • DECODE: decodifica l'istruzione nel registro IR.
  • FETCH DEGLI OPERANDI: preleva i dati necessari per l'operazione.
  • EXECUTE: esegue l'operazione (ad esempio, una somma o una moltiplicazione).

I registri della CPU sono memorie molto veloci che vengono usate per memorizzare temporaneamente i dati e le istruzioni.

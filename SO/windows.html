Windows

Storia di Windows

Microsoft Windows è una famiglia di sistemi operativi prodotta da Microsoft Corporation. nel 1984 la Microsoft annuncia che sta sviluppando Windows, un'interfaccia grafica per il suo sistema operativo MS-DOS che era venduto con i PC IBM e compatibili dal 1981.

Prime Versioni di Windows

Nel 1985 Windows 1.0, ha scarso successo, non fornisce un sistema operativo completo, ma era un'estensione dell'MS-DOS. Nel 1990 Windows ottiene un deciso successo con la nuova versione 3.0, che oltre a funzionalità migliorate date alle applicazioni native, consente l'utilizzo dei programmi di MS-DOS con un multitasking migliore.

Collaborazione con IBM

Con il suo successo Windows 3.0 diventa la maggiore fonte di reddito per la Microsoft, e guida le scelte e i piani successivi dell'azienda. Windows inizia la sua collaborazione con IBM insieme continuano a fare ricerche su OS 2 e arrivano al Windows NT.

Windows NT

Windows NT viene rilasciato definitivamente attraverso Windows 3.1 NT ma non ebbe molto successo a causa delle sue problematiche, era troppo potente per i PC dell'epoca. Tuttavia, le sue caratteristiche lo resero perfetto per il mercato di server LAN che in quel periodo stava decollando, permettendo a Microsoft di ottenere un altro successo.

Multitasking

Il sistema operativo è di tipo multitasking, una funzione di Windows che permette di utilizzare più desktop contemporaneamente. Windows è sin dall'origine un sistema operativo grafico, pensato per essere utilizzato con il mouse e, a differenza dei sistemi Unix, l'interfaccia grafica è un componente essenziale.
